# Flynn Taxonomy - Parallel Computing (Matrix Based)

## Models Implemented:
- SISD (Sequential Matrix Processing)
- SIMD (Single Instruction Multiple Data)
- MISD (Matrix Pipeline Processing)
- MIMD (Parallel Matrix Multiplication using Goroutines)

## Language
Go (Golang)

## Description
This project demonstrates Flynn's Taxonomy using matrix-based computation,
including both sequential and parallel implementations.
